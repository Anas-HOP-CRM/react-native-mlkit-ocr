# react-native-mlkit-ocr

Google on-device ML kit for RN

## Installation

```sh
npm install react-native-mlkit-ocr
```

## Usage

```js
import MlkitOcr from "react-native-mlkit-ocr";

// ...

const result = await MlkitOcr.multiply(3, 7);
```
